# Fitness Tracker API

This is a REST API for tracking fitness activities and goals, built with Django and Django REST Framework (DRF). It allows users to register, log activities (e.g., running, cycling), set weekly goals, and track their progress.

## Project Structure
- `activities/`: App for managing fitness activities.
- `goals/`: App for managing weekly goals and progress.
- `users/`: App for user registration and authentication.
- `fitness_tracker_api/`: Main project directory with settings and URLs.
